# 内训形式

---

除了 lecture 之外（计划中只有第一周有 lecture），其他的都由同学分享为主

我们希望的是，分享的同学可以简要的做一个 ppt，并且将他的技术文档上传到一个 github 仓库

我们教学部的工作是对接好分享的同学？

# 关于心态

即使冷场也不要害怕，协会是大家共建的，内训应当是人人参与，而不是我们教学部负责同学的一厢情愿的付出，所以我们只需要尽量组织大家，如果实在没人参与也不必自责。
